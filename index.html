Git branch tutorial

Git branch makes collaboration easier

To see how many branch type "git branch"

Selected branch has "*" sign attached to it

To make a new branch type "git branch branch_name"

To change from one branch to another type "git checkout branch_name"

To merge a branch to master type "git merge branch_name"

To see difference between two branch type "git diff branch_name_1..branch_name_2"

To remove or delete a branch type "git -d branch_name"
